###  Film Revenue Analytics (DVD Rental Dataset)

This query demonstrates multi-step analytical reporting using Common Table Expressions (CTEs).  
It calculates:
-  number of actors per film,
-  total revenue generated by each film,
-  overall revenue across the entire dataset,
-  percentage share of revenue per film.

Techniques used:
- CTE chains
- aggregations (SUM, COUNT)
- COALESCE for handling missing values
- CROSS JOIN for global metrics
- ordering and ranking

This serves as a practical example of intermediate SQL skills for analytics and reporting.
