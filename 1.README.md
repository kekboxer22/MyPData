This repository contains practical SQL exercises and analytics queries using the **DVD Rental** database.  
It demonstrates skills in CTEs, subqueries, window functions, UNION, CASE, and data aggregation.

## File Overview

### 1. `CTE.sql`
 Comprehensive use of Common Table Expressions (CTEs)
Calculates:
 Actor count per film
  Film revenue
 Total database revenue
 Percentage contribution of each film


### 2. `UNION_CTE.sql`
- Shows use of `UNION ALL` for categorizing films
- Separates films by rating: G vs Non-G
- Uses aggregated revenue for filtering

### 3. `case.sql`
- Demonstrates conditional aggregation with `CASE`
- Example: Revenue classification into "Top", "Mid", "Low"

### 4. `clean_CTE.sql`
- Cleaned and optimized version of the CTE queries
- Easier to read and maintain

## Notes
- All queries are tested on PostgreSQL / DVD Rental DB
- Commented thoroughly for learning and demonstration purposes
- Designed to show analytical thinking and SQL skills
